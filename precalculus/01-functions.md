## 1.1 Functions and Function Notation
### Determining Whether a Relation Represents a Function
**relation** - a set of ordered pairs.  
**domain** - the set of the first components of each ordered pair.  
**range** - the set of the second components of each ordered pair.  
Example:  
{(1,2),(2,4),(3,6),(4,8),(5,10)}  
domain: {1,2,3,4,5}  
range: {2,4,6,8,10}  

**input/independent variable** - each value in the domain. often labeled with the lowercase $x$  
**output/dependent variable** - each value in the range. often labeled with the lowercase $y$  

function $f$ - a relation that assigns a single value in the range to each value in the domain.  
Example of a set that relates "even" and "odd" to the first five natural numbers:  
{(odd,1),(even,2),(odd,3),(even,4),(odd,5)}  
The term "odd" corresponds to three values from the range {1,3,5} and "even" to two values {2,4}. This violates the definition of a function. This relation is not a function.  

Relations that are functions and not functions:  
Example A:  
$p \to m$  
$q \to n$  
$r \to n$  
This relationship is a function because each input is associated with a single output.  

Example B:  
$p \to x$  
$q \to y$  
$r \to z$  
Also a function. Each input is associated with a single output.  

Example C:  
$p \to x$  
$q \to y$  
$q \to z$  
This relationship is not a function because $q$ is associated with two different outputs.  

> **function**  
A **function** is a relation in which each possible input value leads to exactly one output value. We say "the output is a function of the input." The **input** values make up the **domain**, and the **output** values make up the **range**

#### Using Function Notation
There are various ways of representing functions. A standard function notation is one representation that facilitates working with functions.  
To represent "height is a function of age" start by identifying the descriptive variables $h$ for height and $a$ for age.  
$h$ is $f$ of $a$   -   We name the function $f$; height is a function of age.  
$h = f(a)$  -   We use parentheses to indicate the function input.  
$f(a)$  -   We name the function $f$; the expression is read as "$f$ of $a$."  
Any letter can be used to name the function. The notation $h(a)$ shows that $h$ depends on $a$. The value $a$ must be put into the function $h$ to get a result. Parentheses indicate age is input into the function; they do not indicate multiplication.  
We can give an algebraic expression as the input to a function. $f(a+b)$ means first add $a+b$, then use the result as the input of $f$.  

> **function notation**  
The notation $y = f(x)$ defines a function named $f$. This is read as "$y$ is a function of $x$." The letter $x$ represents the input value, or independent variable. The letter $y$, or $f(x)$, represents the output value, or dependent variable.
